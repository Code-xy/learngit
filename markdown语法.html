3. 列表

Markdown 支持有序列表和无序列表。

无序列表使用星号(*)、加号(+)或是减号(-)作为列表标记，这些标记后面要添加一个空格，然后再填写内容：

xxxxxxxxxx
* 第一项
* 第二项
* 第三项
​
+ 第一项
+ 第二项
+ 第三项
​
​
- 第一项
- 第二项
- 第三项
  • 第一项
  • 第二项
  • 第三项
  1. 建设社会主义
  2. 好青年
  3. 河山大好

列表嵌套 列表嵌套只需在子列表中的选项前面添加四个空格即可：

  1. 第一个列表 健身 打卡 旅游
  2. 第二个列表 1 2 3 4

Markdown 区块

Markdown 区块引用是在段落开头使用 > 符号 ，然后后面紧跟一个空格符号：

区块引用 菜鸟教程 学的不仅是技术更是梦想

另外区块是可以嵌套的，一个 > 符号是最外层，两个 > 符号是第一层嵌套，以此类推：

xxxxxxxxxx
> 最外层
> > 第一层嵌套
> > > 第二层嵌套

7.Markdown 表格

Markdown 制作表格使用 | 来分隔不同的单元格，使用 - 来分隔表头和其他行。

语法格式如下：

xxxxxxxxxx
|  表头   | 表头  |
|  ----  | ----  |
| 单元格  | 单元格 |
| 单元格  | 单元格 |

对齐方式

我们可以设置表格的对齐方式：

  • -: 设置内容和标题栏居右对齐。
  • :- 设置内容和标题栏居左对齐。
  • :-: 设置内容和标题栏居中对齐。

实例如下：

xxxxxxxxxx
| 左对齐 | 右对齐 | 居中对齐 |
| :-----| ----: | :----: |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |
